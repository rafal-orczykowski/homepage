Informacje II tygodnia

Style umieszczane są w sekcji <head></head>

Tag {atrubut: wartość;} - stylowanie za pomocą obiektów. Odnosi się do wszystkich elementów o tym samym tagu.

Informacje III tygodnia

<tag style="atrybut:wartość">treść</tag> - Inline style. Styl znajduje się w tagu razem z atrybutem i wartością w pliku html. Odnosi się do pojedynczego przypadku.

<tag class="name">treść</tag> - Nadawanie klasy. W pliku CSS można w ten sposób nadać atrybuty każdemu znacznikowi posiadającemu daną klasę. 'Kotwicą' dla klasy w pliku css jest '.' - .name {atrybut:wartość;}

<tag id="name">treść</tag> - Nadawanie id. W pliku CSS można w ten sposób nadać atrybuty znacznikowi posiadającemu dane id. 'Kotwicą' dla id w pliku css jest '#' - #name {atrybut:wartość;}. Jedno id powinno być przypisane tylko jednemu znacznikowi.

Nazwy klas oraz id zapisujemy kebab-casem.

Do wyszukiwania elementów w css można używać selektorów np. article h3 {atrybut:wartość;}. Zostanie wyszukany h3 znajdujący się w article. Element ścieżki nie musi być bezpośrednim 'rodzicem' znacznika do którego się odnosimy.

Należy pamiętać o Specyficzności CSS. Kolejność zczytywania wartości jest następująca: important!, style, Id, class, element. NIE ZAGNIEŻDŻAMY!
